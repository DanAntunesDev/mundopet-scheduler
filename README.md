# MundoPet Scheduler

Sistema de agendamento para pet shop em desenvolvimento, criado com foco em organização de horários, separação de responsabilidades no código e evolução progressiva da aplicação.

## Status do projeto

🚧 Em desenvolvimento

Este repositório está sendo construído de forma incremental, com commits frequentes, melhorias contínuas e evolução da estrutura conforme o projeto avança.

## Objetivo

Praticar e consolidar conceitos como:

- organização de projeto em módulos
- separação de responsabilidades
- manipulação de dados de agendamento
- estruturação de regras de negócio
- evolução prática de sistemas JavaScript

## Tecnologias utilizadas

- JavaScript
- Vite
- HTML
- CSS

## Funcionalidades atuais

- exibição de agendamentos
- organização dos horários por período
- estrutura modular do projeto
- evolução contínua da regra de negócio

## Próximos passos

- melhorar o fluxo de criação de agendamentos
- refinar regras de negócio
- ampliar a camada de serviços
- preparar integração com backend real
- melhorar a experiência de uso da aplicação
